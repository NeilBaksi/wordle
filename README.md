# Wordle 🟩

I built this for my mum — she loves Wordle and deserves her own version. 💚

## 🛠️ Tech Stack

- **React 19** + **TypeScript**
- **Vite 8**
- **Tailwind CSS v4**
- Hosted on **Vercel**

## 🎮 Features

- Daily word + unlimited replay mode
- Hard mode 💪
- Hint system 💡 (one per word)
- 3 themes: Dark · Light · Sketchbook
- Fully responsive
